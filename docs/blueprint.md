# **App Name**: Kanak's AI Portfolio

## Core Features:

- Dynamic Hero Section: Eye-catching 3D animations and clear value proposition to immediately impress visitors in the hero section.
- Interactive Project Showcase: Use hover effects, detailed descriptions, and 3D animations to highlight the impact of each project, optimized for ATS.
- AI-Powered Content Tool: Uses AI tool to generate alternative headline and bio variants, allowing the user to A/B test them for effectiveness and engagement, and presenting these alternatives in an easy-to-use UI.
- Experience Timeline: Visual timeline highlighting relevant experience and achievements in a clear, recruiter-friendly format.
- Contact Form & Social Links: Easy-to-use contact form with direct links to LinkedIn, GitHub, and email to encourage connections.
- Mobile Responsive Design: Ensures the portfolio looks and functions flawlessly across all devices, enhancing accessibility.

## Style Guidelines:

- Primary color: Teal (#008080) to evoke professionalism and innovation.
- Background color: Off-white (#F5F5F5) to provide a clean and modern aesthetic.
- Accent color: Deep Blue (#000080) to highlight key elements and add a touch of sophistication.
- Font: 'Poppins' (sans-serif) for a sleek, modern, and readable interface.
- Use minimalist, professional icons to complement the clean theme. Examples include line-style network symbols and stylized data icons.
- Grid-based layout with rounded card edges and subtle shadows to create depth and a clean, organized look.
- Subtle hover effects and highlights on skills to engage users and draw attention to key areas of expertise.
- 3D animations in the hero section and on each project to add visual interest and engagement.
